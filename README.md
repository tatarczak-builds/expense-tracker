# Expense Tracker 
 
A simple command-line expense tracker built to practice Python programming.


## Features

- Add expenses 
- Add multiple expenses in one session
- Remove chosen expenses
- Remove multiple expenses in one session
- Display all expenses
- Display the total amount of expenses
- Save and load expenses from a file
- Handle invalid input


## Technologies

- Python


## How to run

1. Clone the repository
2. Navigate to the project directory
3. Run:

```bash
python system.py
```


## Project Structure

- `system.py` - main program and menu
- `functions.py` - expense management functions
- 'expenses.txt' - stores saved expenses
