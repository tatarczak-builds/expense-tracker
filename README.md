# Expense Tracker 
 
Simple command-line program built to practice Python

## Features

- Add expenses 
- Add multiple expenses in one session
- Displaylist and total of expenses
- Remove chosen expenses
- invalid menu input

## Technologies

- Python
- Functions
- Modules
- Loops
- Exception handling

## What I learned

- Working with lists and tuples
- Basic error handling 
- Proper use of enumerate() and sum()
- The difference between `append()` and `pop()`

## Project Structure

- `system.py` - main program and menu
- `functions.py` - expense management functions
